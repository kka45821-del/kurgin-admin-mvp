# KURGIN Admin — Stage 01 Import

Рабочий пакет Этапа 1: безопасный импорт поставки.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Что умеет Этап 1

- загрузка готового Excel KURGIN Score Result;
- ввод данных поставки;
- чтение листов `Results`, `Details`, `Issues`, `System`;
- предпросмотр импорта;
- отмена без сохранения;
- подтверждение импорта;
- backup перед записью;
- сохранение `stones_master.csv`, `shipments.csv`, `import_log.csv`;
- сохранение raw-слоя по `import_id`;
- сохранение оригинального Excel как `original.xlsx`;
- просмотр камней поставки без редактирования.

## Что не входит в Этап 1

- редактирование камней;
- цены;
- публикация;
- экспорт публичного каталога;
- полноценная обработка конфликтов ID;
- PostgreSQL;
- сайт.
