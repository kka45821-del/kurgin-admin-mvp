# IMPORT_FLOW.md

## Этап 1 — безопасный импорт поставки

Цель:

```text
Excel → данные поставки → предпросмотр → подтверждение → сохранение
```

## Excel

Загружается готовый KURGIN Score Result Excel.

Обязательные листы: `Results`, `Details`, `Issues`, `System`.

Если файл не читается как Excel или отсутствует обязательный лист, импорт блокируется.

## Данные поставки

Обязательные поля: Поставщик, Дата поставки, Название / номер поставки, Валюта, Стоимость закупки всей партии, Аванс.

Необязательные поля: Комментарий по оплате, Общий комментарий.

Не вводятся отдельными полями: Остаток к оплате, Статус оплаты. Остаток считается автоматически.

## import_id / shipment_id

В первой версии `import_id = shipment_id`.

Формат: `IMP-ГГГГММДД-001`, например `IMP-20260604-001`.

## До подтверждения

До подтверждения импорта ничего не записывается в постоянные файлы. Не создаются изменения `stones_master.csv`, `shipments.csv`, `import_log.csv`, raw-папка, `original.xlsx`, частичные CSV, exports.

Отмена = ноль следов в постоянных файлах.

## Предпросмотр

Показывает данные поставки, имя файла, версии и System, количество найденных камней, сколько будет сохранено, сколько не будет сохранено, камни вне текущей версии, ROUND/не ROUND, main/large, Issues, конфликты ID, отсутствующие важные поля, итоговое действие по каждой строке.

Интерфейс компактный: сводка сверху, раскрываемые блоки/таблицы ниже.

## Кнопки

После предпросмотра: «Подтвердить импорт», «Отменить импорт», «Назад / изменить данные поставки».

## После подтверждения

Создаётся backup, сохраняются `stones_master.csv`, `shipments.csv`, `import_log.csv`, raw-слой и `original.xlsx`.

## Какие камни сохраняются

Сохраняются только камни первой версии:

```text
main: от 1.00 ct включительно до 3.00 ct не включительно
large: от 3.00 ct включительно и выше
```

Камни других будущих разделов показываются в предпросмотре, но не сохраняются.

Не ROUND камни от 1.00 ct и выше сохраняются, если входят в main или large. Они идут без KURGIN Score.

## Начальные значения

Все сохранённые камни получают:

```text
status = draft
availability_status = in_stock
```

Импорт не равен публикации.

## После успешного импорта

Показать сообщение «Поставка успешно импортирована», номер поставки/import_id, количество сохранённых и не сохранённых камней.

Кнопки: «Перейти к камням этой поставки», «Открыть список поставок», «Загрузить новую поставку».

На Этапе 1 экран «Камни этой поставки» только для просмотра.
