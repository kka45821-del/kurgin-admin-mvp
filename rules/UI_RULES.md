# UI_RULES.md

## Язык интерфейса

Интерфейс `kurgin-admin` по умолчанию должен быть на русском языке.

На русском должны быть:

```text
названия страниц
кнопки
подсказки
предупреждения
сообщения ошибок
сообщения подтверждения
предпросмотры импорта
предпросмотры публикации
отображаемые статусы
```

Английские технические ключи могут храниться внутри данных и кода, но в интерфейсе показываются понятные русские названия.

Примеры:

```text
status = draft → Черновик
status = published → Опубликован
availability_status = in_stock → В наличии
availability_status = reserved → Забронирован
catalog_section = main → Основной каталог
catalog_section = large → Крупные камни
published_price → Публичная цена
```

## Компактность интерфейса

Админка должна быть компактной и рабочей, без лишних больших отступов.

Правила:

```text
плотные формы
минимум пустого пространства
компактные таблицы
короткие подписи
аккуратные блоки
сводка сверху
детали ниже в раскрываемых блоках или таблицах
```

Компактность особенно важна для:

```text
Загрузка поставки
Предпросмотр импорта
Камни этой поставки
Поставки
Таблица камней
Цены
Публикация
Экспорт
```

## Без потери ясности

Компактность не должна скрывать важные предупреждения.

Для опасных действий всегда нужны:

```text
предпросмотр
понятное предупреждение
подтверждение
backup перед записью
```
