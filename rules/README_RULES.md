# KURGIN Admin Rules — Stage 02

Этап 2 — Управление камнями.

Раздел «Камни» = главный реестр всех сохранённых камней.

Включено:

- фильтры;
- поиск;
- компактная таблица;
- карточка камня;
- редактирование только административных полей;
- raw Results / Details / Issues в карточке.

Не входит:

- полная система цен;
- публикация;
- экспорт;
- редактирование геммологических данных;
- клиентская карточка товара.
