# KURGIN Admin Rules — Stage 01 v2-delete-fields

Этап 1 включает:

- безопасный импорт поставки;
- сохранение raw-слоя;
- поля `fluorescence`, `min_diameter`, `max_diameter`, `depth_mm`;
- минимальное безопасное удаление конкретной поставки полностью.

Удаление поставки выполняется только через:

```text
предпросмотр → ввод номера поставки → backup → удаление
```

Этап 9 остаётся для сложной архивной очистки и исторических удалений.

Streamlit Cloud runtime не считать постоянным хранилищем настоящих данных.
